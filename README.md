Agencia de Viajes

Proyecto web desarrollado en PHP, HTML, CSS, JavaScript y MySQL, que permite la gestión de usuarios, reservas, pagos, login, registro y visualización de información turística. Este repositorio contiene todo el código fuente del sistema.

Características principales:
Sistema de autenticación (login, registro y cierre de sesión)
Gestión de reservas de viajes
Panel de usuario para ver y modificar datos personales
Registro de viajes con detalles y confirmación
Proceso de checkout en múltiples pasos
Generación de facturas en formato HTML
Manejo de imágenes, estilos y scripts organizados por carpetas

Tecnologías Utilizadas:
PHP 7+
MySQL / MariaDB
JavaScript
HTML5 / CSS3 / SCSS
Bootstrap
XAMPP como entorno local
